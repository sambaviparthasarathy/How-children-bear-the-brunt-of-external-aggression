# How-children-bear-the-brunt-of-external-aggression
The Minab school strike in Iran and mounting death toll in Lebanon serve as a stark reminder of how children are at an increasing risk of becoming 'collateral damage' in escalating conflicts. This data set consists confirmed events of external attacks that resulted in a child fatality.
Methodology: 
1. I analysed daily event data from ACLED only for inter-state conflict data where States are external actors. I used coding using AI studio to calculate events with child fatalities. ACLED’s daily conflict data was downloaded from 1996-2026. Only 4 interaction codes were used for the analysis: 18, 88, 78, and 28
● 18: STATE FORCES VERSUS EXTERNAL/OTHER FORCES (e.g., inter-state conflict; state engagement with private security forces or a UN operation; strategic developments between a regime and the UN or another external actor)
● 88: OTHER VERSUS OTHER (e.g., clashes between foreign state forces, international missions, or private security forces)
● 78: OTHER ACTOR VERSUS CIVILIANS (e.g., regional or international operation targeting civilians; private security forces targeting civilians)
● 28: REBELS VERSUS OTHERS (e.g., civil war violence between rebels and an allied state military; rebel violence against a UN operation) Eg. Hamas, Houthis are rebel groups. Category pertains to conflict between states and these rebel groups.
2. Code was used to filter interaction codes and read through notes to count events each year.
3. Events had to be counted from descriptions that read like these: child killed, teen shot dead, 10-year-old killed, women and children killed etc..mapped all possible caveats in this step.
4. Then, Actor 1 like military forces of Israel etc was mapped to countries (in this case, Israel) and then code was used to draw pivot tables and later analyse in the excel sheets.

Limitations of the analysis:
Only inter-state conflicts, ignores civil wars etc.
Pertains only to frequency of attacks that resulted in at least one child casualty
Doesn’t count fatalities (Bcos In most of the notes, fatalities are counted jointly: Example: over 8 women and children killed…. ). Real number could be much higher.
Based on ACLED’s conflict data which is based on reports etc. Could possibly undercount events
Fatalities could be higher. One attack could have killed 50 children, while another could have killed just one child. Both will be counted as events of attack.
